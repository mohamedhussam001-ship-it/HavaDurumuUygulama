# HavaDurumuUygulama

HavaDurumuUygulama projesi, kullanıcıların belirli şehirler için gerçek zamanlı hava durumu bilgilerine erişmesini sağlayan, harici bir hava durumu API’si ile entegre çalışan, alınan JSON verilerini işleyerek sıcaklık, nem, rüzgar hızı ve genel hava durumu gibi temel meteorolojik bilgileri basit ve kullanıcı dostu bir arayüz üzerinden sunan, böylece RESTful API iletişimi, veri ayrıştırma ve dinamik kullanıcı arayüzü geliştirme süreçlerinin modern uygulama geliştirme ortamında pratik olarak uygulanmasını gösteren bir mobil uygulama projesidir.
